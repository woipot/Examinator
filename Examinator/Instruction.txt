Examinator/Examinator/Instruction.txt
    
Данная программа разработа для создания тестов и прохождения тестов.

Дополнительной сложностью для студентов является то, что может быть 
несколько правильных вариантов ответа, так что для того, чтобы задание засчиталось 
как выполненное, необходимо выделить ТОЛЬКО ВСЕ ПРАВИЛЬНЫЕ ответы.
(Если студент выберет не все правильные 
или хотя бы один лишний неверный ответ, оценка за задание = 0 баллов)

Результы находятся в папке Result (зашифрованы)
Тесты находятся в папке Tests (зашифрованы)

(!) При изменении файлов результатов и тестов вне программы, доступ к ним будет невозможен;

////////////////////////////

Перенос тестов на другой компьютер:

1. Копируете необходимые тесты (название_теста.xml) из папки Tests на флеш накопитель
2. Переносите их в такую же папку Tests, только на другом компьютере

МЕНЮ:

===Режим преподавателя:
	включает дополнительные функции, такие как:
редактирование, удаление и создание тестов;

===Создать тест (ВИДНА ТОЛЬКО В РЕЖИМЕ ПРЕПОДАВАТЕЛЯ, далее РП):
	Подготавливает и открывает пустой тест в редакторе, тест не будет сохранен 
до тех пор, пока не будет нажата кнопка "Сохранить" с последующим сообщением об успехе

===Загрузить тест (РП): 
	Позволяет загрузить тест из текстового файла (.txt).
Подразумевается, что тесты в формате .txt (то есть не зашифрованные файлы)
не хранятся на пользовательских компьютерах, а хранятся
исключительно на компьютере преподавателя. Впоследствии текстовые файлы должны быть удалены,
чтобы вопросы не хранились в незашифрованном виде.

	Пример входного текстового файла (.txt):
	
Название = Пробный тест
Автор = Иванов Иван
Время на вопрос = 60  * Указывается в секундах
Вопросы = 2
Можно пропускать = нет  * Можно выбрать 'нет' или 'да'

=Какого цвета солнце?
+белое
-желтое
-голубое

=Сколько будет 2+2*5? 
-20
-4
+12

В файле могут отсутсвовать такие значения параметров: 
Автор = ...
Можно пропускать = ...

Если не указано значение для пропускания, то тест будет с пропускаемыми вопросами,
то есть стандартное значение: Можно пропускать = да

Вопросы начинаются со знака "=" равно,
а ответы к ним: "-" минус, если ответ неверный
		"+" плюс, если верный

В каждом вопросе должен быть хотя бы один "+" (Можно больше).

	Вопросы = ... Количество вопросов, которые будут случайным обазом выданы студенту
из общего числа вопросов (Минимальное значение = 1)


	Время на вопрос = ... указывается в секундах (на один), столько времени будет выделено
для студента на один вопрос. Если стоит значение (Можно пропускать = да) время теста 
будет = Время на вопрос * Вопросы.  (Минимальное значение = 10 [10 сек на один вопрос])
/////////////////

РЕДАКТОР (РП) (кнопка ""):
ПОСЛЕ ВСЕХ ИЗМЕНЕНИЙ, НЕОбХОДИМО НАЖАТЬ КНОПКУ "Сохранить"!
Тест будет перезаписан в зашифрованный файл, ПЕРЕЗАПУСК ПРОГРАММЫ НЕ ТРЕБУЕТСЯ!

Если в ходе редактирования были допущены ошибки, программа сообщит 
о них в виде окна со списком ошибок (Критические ошибки начинаются со знака *), Например:
	*<Отсутствует название теста>
После устранения ошибок, повторно нажмите кнопку "Сохранить".

Сохранение гарантировано только в случае появления окна с надписью
об успешном сохранении.
////////////////////////////

ТЕСТИРОВАНИЕ c возможностью пропускать вопросы:

В программе существует возможность пропускать вопросы, то есть делать их не по порядку.

1 случай: если поставить галочку напротив слов "можно пропускать":
	Можно пропускать = да 
	Тогда студент решает вопросы в произвольном порядке до истечения сумарного времени на тест.

2 случай: если не поставить галочку напротив слов "можно пропускать":
	Можно пропускать = нет 
	Тогда студент решает вопросы по порядку и имеет ограниченное время на каждый вопрос.

Вопросы перемешиваются, также как и ответы внутри вопросов.
Отвеченные вопросы отмечаются определенным цветом.

Критерии оценивания:
100%:      оценка 5
80% - 99%: оценка 4
70% - 79%: оценка 3
0-69%:     оценка 2
